Project Overview

This project implements a complete DevOps pipeline for the lifecycle of an application, from building the code to deploying it on Kubernetes using Docker containers. The pipeline automates CI/CD, security scanning, monitoring, and more.

DevOps Tools Used

  • Git & GitHub
  • Jenkins for CI/CD
  • Docker
  • Kubernetes
  • Helm
  • Prometheus & Grafana (Monitoring)
  • SonarQube (Code Quality & Security)
  • Nexus (Artifact Repository)
